# Frontend (Vue 3 + Vite) — v0.0.2 MVP

Минимальный UI: Login, Register, Tasks. Работает с backend по `VITE_API_BASE_URL` (по умолчанию `http://localhost:8000/api`).

## Запуск
```
cd frontend
npm install
npm run dev
```

Создайте `.env` при необходимости:
```
VITE_API_BASE_URL=http://localhost:8000/api
```

## Тесты (Vitest)
```
npm run test
```

Через just:
```
just test
```
