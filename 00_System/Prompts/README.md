# Prompts для агента

Здесь лежат “промпты‑роли” для AI‑агента, который работает напрямую с файловой системой и Git.

## Как использовать
1. Всегда начинай с прочтения: `AGENTS.md` (правила репозитория).
2. Затем выбери одну роль под запрос пользователя:
   - `00_System/Prompts/Assistant.md`
   - `00_System/Prompts/Coach.md`
   - `00_System/Prompts/Repo-Maintainer.md`
   - (первый запуск) `00_System/Prompts/Bootstrapper.md`
3. Если запрос смешанный — используй `00_System/Prompts/Router.md` (разбивка на под‑задачи).

## Принцип
Промпты в этой папке — это “стабильные инструкции”, а не разовые задачи. Они должны меняться редко и маленькими диффами.
