# Informika — Ф3 tech spec (выжимка)

## Summary
- Источник: `C:\informika\docs\tech-spec-F3.md` (Ф3: публичный портфель/кейсы, события, аналитика, экспорт Excel).

## Notes
- Включено (крупно): публичные страницы идей/кейсов (только просмотр), публичный календарь событий + регистрация и e-mail уведомления, минимальная аналитика по ролям + экспорт Excel, каталог команд/результатов (по одобренным идеям/опубликованным кейсам), надёжность с кэшированием данных Bitrix24.
- Портфель проектов:
  - вкладки: `Одобренные идеи` (только статус `Одобрена`) и `Кейсы` (только опубликованные кейсы).
  - публичная страница проекта (одобренная идея): только просмотр, без управления командой/решений.
  - кейс создаётся только после архивации, публикация кейса не меняет статус в Bitrix24.
- События:
  - статусы: `Черновик` → `Опубликовано` → `Завершено/Отменено` → `Архив`.
  - публичный календарь: виды `Календарь (месяц/неделя)` + `Список`, фильтры `Будущие/Прошедшие`, `Формат`, `Направление`, `Поиск`.
  - регистрация гостя: ФИО, e-mail, роль + согласие на ПДн.
  - уведомления: подтверждение регистрации, напоминания за 24ч/2ч, материалы после события.
- Аналитика:
  - публичный блок “За 7 дней” на странице “Банк инноваций” (без персональных данных);
  - участник: “Мой вклад”;
  - стейкхолдер: блоки активности/прогресса/рисков (данные Bitrix24);
  - руководство: сводка + экспорт Excel.
